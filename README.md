# Sheryians Coding School - Cohort 3.0: Assignment 1
## Pokémon Gen I UI Landing Page

**[🚀 View Live Demo Here!](https://nojan-devkota.github.io/pokemon-page/)**

### 📝 Overview
This project is part of **Assignment 1** for the **Sheryians Coding School Cohort 3.0**. It is a modern, responsive split-screen landing page featuring Pikachu from Pokémon Generation I. The project focuses on translating a reference design into a functional web layout using pure HTML and CSS.

### 🚀 Technologies Used
- **HTML5**: Semantic markup and structure
- **CSS3**: Styling, Flexbox, and absolute positioning
- **FontAwesome**: For UI icons (hamburger menu)

### 💡 Key Concepts Demonstrated
Throughout the creation of this project, several core front-end concepts were applied and mastered:
- **Flexbox Layouts**: Creating a flexible split-screen design (`.left` and `.right` panels) without hard-coded heights, naturally preventing scrollbar overflows.
- **Absolute Positioning**: Layering UI elements (like the navigation bar and vertical slider dots) exactly where they need to be, floating perfectly above the background colored columns.
- **Typography & Alignment**: Utilizing custom fonts, adjusting `letter-spacing`, and applying negative margins to correct built-in font bearing issues for pixel-perfect visual alignment.
- **Responsive Aspect Ratios**: Scaling large hero images seamlessly using auto-calculated dimensions so they maintain their native proportions.

### 📂 File Structure
- `index.html` - The main DOM structure of the webpage.
- `style.css` - All of the custom styling, positioning, and layout rules.
- `Images/` - Directory containing the Pokémon logo, Pikachu hero image, and Pokeball background graphic.

### 🏃‍♂️ How to Run
Simply download or clone the repository, and double-click the `index.html` file to open it in any modern web browser (Google Chrome, Firefox, Safari, Edge). No build steps or servers are required!
